# Big Systems OS


## Prerequisites (For Build)
* GCC
* make
* VM Software (such as QEMU)
* Linux, WSL or Docker 

## How to Build
```bash
$ cd BigSystemsOS
$ make build-x86_64
```

## Emulate
### Linux
```bash
$ make run
```

### Windows
run the file `src/kernel/src/run2.bat`
